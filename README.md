# Internal Security Audit & Risk Assessment: Botium Toys

## Project Overview
This project consists of a comprehensive security audit and risk assessment conducted for "Botium Toys," a retail company. The audit evaluates the organization's existing assets, identifies critical control vulnerabilities, and reviews adherence to major compliance regulations (PCI DSS, GDPR, and SOC).

## Objectives
* Assess corporate and IT assets to identify security gaps.
* Evaluate organizational practices against the NIST CSF framework.
* Complete a controls and compliance checklist to establish baseline vulnerabilities.
* Provide risk-mitigation recommendations to align business practices with security standards.
